# 影音處理與製作

從 YouTube 內容製作到 Podcast 混音，累積超過 70 支完整製作影片與 180+ 小時觀看時間。建立了從編曲、錄音、混音、拍攝、字幕到發行的標準化製作流程。

## 主要項目

### YouTuber 與內容製作 🎬

經營 BloggerMandolin YouTube 頻道，每支影片都是完整的 End-to-End 製作。

**製作流程：**
- 70+ 支完整製作影片（錄製、混音、剪輯、字幕、發行）
- 180+ 小時累積觀看時間
- 從編曲到發行的標準化 SOP
- 影片拍攝、剪輯、字幕、後期特效

**製作工具：** Premiere Pro / After Effects / Logic Pro X

---

### 觀念碰撞所 Podcast 混音師 🎙️

擔任觀念碰撞所 Podcast 混音師，負責音訊混音編輯與品質把關。

**工作內容：**
- 音訊混音編輯與後期處理
- 確保播客音質專業度與聽眾體驗
- 訊號處理（EQ / 動態控制 / 降噪）
- 維持穩定的音質輸出標準

**混音工具：** Logic Pro X / 混音 / 母帶後期 / 訊號處理

---
