# 工具與應用程式

這是我大一時期（2022 年 2 月）開發的兩個 Python 小工具，用來解決自己生活中的實際需求。

## 主要專案

### [Discord 訊息推播機器人](./discord-bot.md) 🌌

由於不喜歡打開社群媒體的通知系統（會跳出廣告資訊或無關的訊息），但仍有一些資訊想立即知道，因此製作了這個訊息推播機器人。

**核心功能：**
- 整合 Gmail、RSS、YouTube、Twitch、加密貨幣價格追蹤
- 用 requests + BeautifulSoup 爬取 YouTube 社群貼文
- 統一推播到 Discord 私人伺服器
- 在 Replit 上長期運行，減少社群使用時間

**開發時間：** 2022 年 2 月 | **狀態：** 已完成（不再維護）

---

### [GoodTime 番茄鐘記錄分析工具](./time-tracker.md) 🍅

從高中開始使用番茄鐘軟體記錄時間履歷，為了定期檢視自身時間規劃，製作了這個分析工具。

**核心功能：**
- 解析 Goodtime Productivity 匯出的 CSV 檔案
- 用 Matplotlib 生成效率分析圖表
- 統計工作項目的累積時間和平均值
- Tkinter GUI 介面提供簡單操作

**開發時間：** 2022 年 2 月 | **狀態：** 已完成（不再維護）
